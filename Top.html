How to customize rotation!
And other Feedback and Requests I hope you find this App useful.
If you wish to customize the rotation, or have a request for a new feature, or find something not working quite right, or to just to let me know you are using it, leave feedback in form below or email to simard57@hotmail.com.
The rotation schedule is generated from a spreadsheet. To change the rotation:
  1. Download the Schedule.csv for the Day and Shift that you want to change from Rotation Schedules
  2. Edit the Schedule.csv file using any Spreadsheet program you have
  3. VERY IMPORTANT! Save the Schedule.csv file as a .csv! The online Excel & Google Sheets save in their native formats.
  4. Upload the updated Schedule.csv in the appropriate Day and Shift at Updated Rotation Schedules
  5. When I notice the file upload, I will update the website with your new rotation. You can ping me using feedback form below for quicker response.
